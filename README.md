# Sonic Odyssey Bot

## Clone the Repository

To get started, clone the repository using the following command:

```bash
git clone https://github.com/0xenesZ/sonic-odyssey-bot.git

````
than

```bash
cd sonic-odyssey-bot
````

than

```bash
npm install
````

```bash
touch privateKeys.json
````
